# 관리자 패치 추가/삭제 기능

## 개요

- **작업일**: 2026-01-16
- **브랜치**: feature/38
- **상태**: 완료

## 목표

관리자 캐릭터 상세 페이지에서 패치 추가/삭제 기능 구현

## 현재 상황

- 패치 수정만 가능 (`PATCH` API + "수정" 버튼)
- 패치 추가/삭제 기능 없음

## 구현 내용

### 1. API 엔드포인트 추가

- `DELETE /api/admin/characters/[name]/patches/[patchId]` - 패치 삭제
- `POST /api/admin/characters/[name]/patches` - 새 패치 추가

### 2. UI 컴포넌트 수정

- `AdminPatchList.tsx`
  - "패치 추가" 버튼 추가
  - 각 패치에 "삭제" 버튼 추가
  - 삭제 확인 모달 추가
- `PatchEditForm.tsx`
  - `patch: null`일 때 추가 모드로 동작
  - 추가 모드에서 패치 ID, 버전, 날짜 입력 필드 표시

## 변경 파일
