# 패치노트 실험체 목록 추출

## 작업 정보

- **날짜**: 2026-01-11
- **브랜치**: feature/36
- **상태**: 완료

## 목표

Firebase에 등록된 모든 패치노트에서 실험체 목록을 추출하여 신뢰할 수 있는 기준 데이터 생성

## 배경

- 기존 패치 데이터 무결성 점검 결과가 신뢰하기 어려운 상황
- 패치노트 HTML에서 직접 실험체 정보를 추출하여 검증 기준 마련

## 작업 내용

1. Firebase에서 모든 패치노트 ID 조회
2. 각 패치노트 HTML에서 `<h5>실험체</h5>` 섹션 찾기
3. 해당 섹션 내 `<strong>` 태그에서 실험체 이름 추출
4. VALID_CHARACTERS 목록으로 필터링
5. 패치별 실험체 목록 JSON으로 저장
